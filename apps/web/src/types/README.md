# Types

## 📌 Purpose
Contains TypeScript type definitions and interfaces used throughout the Keeper platform.

## 🧱 Key Files
- `kip.ts` - Types for KIP (Keeper Interface Protocol) system including agents and intents

## 🔄 Data & Behavior
Defines the type structure for:
- KIP command intents and agent information
- Agent registry and orchestration types
- Thought processing and priority types
